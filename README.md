### Building from source
Requirements to run:
- [just](https://github.com/casey/just?tab=readme-ov-file#pre-built-binaries)
- [cargo/rust](https://doc.rust-lang.org/cargo/getting-started/installation.html)
- [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)
- [typescript](https://www.typescriptlang.org/download/)
- [trunk](https://trunk-rs.github.io/trunk/guide/getting-started/index.html)

In a terminal, run
```
git clone https://github.com/Terpsichord/nea-website 
```
to download the project contents.

To run the web app, run:
```
just full-run
```
at the root of the project.

Alternatively, run:
```
just editor
```
to run the desktop app.
